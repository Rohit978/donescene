---
name: ds-new-milestone
description: Create a new milestone with phases in the roadmap.
---

<objective>
Add a new milestone to `.ds/ROADMAP.md` and break it into sequential phases.
</objective>

<process>
1. Read `.ds/ROADMAP.md` to see existing milestones.
2. Ask the user what the new milestone should accomplish.
3. Decompose the milestone into 3-6 sequential phases.
4. Append the new milestone and its phases to `.ds/ROADMAP.md`.
5. Notify the user to run `/ds-plan [phase_number]` to plan the first phase.
</process>
